Lancement du projet (résumé)

  3. Backend : terminal 1 → cd web_app\backend → créer/activer le venv → pip install -r requirements.txt → python app.py (serveur sur http://127.0.0.1:5000).
  4. Frontend : terminal 2 → cd web_app\frontend_react → npm install → npm run dev (application sur http://localhost:5173).
  5. Connexion : admin@ecoagadir.ma / Admin2024! (détails dans SETUP.md).
